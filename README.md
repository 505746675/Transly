\# Transly - AI 翻译工具



!\[Transly](https://img.shields.io/badge/Transly-AI翻译工具-blue)

!\[版本](https://img.shields.io/badge/版本-1.0.0-green)

!\[许可协议](https://img.shields.io/badge/许可协议-MIT-lightgrey)



Transly 是一个现代化的 AI 翻译工具，结合文本翻译和图片 OCR 识别翻译功能，提供简洁优雅的用户体验。



\## ✨ 功能特性



\- \*\*文本翻译\*\*: 支持超过 25 种语言的互译

\- \*\*图片翻译\*\*: 上传图片自动识别文字并翻译

\- \*\*智能语言检测\*\*: 自动检测源语言

\- \*\*拖拽/粘贴支持\*\*: 支持拖拽图片和从剪贴板粘贴图片

\- \*\*响应式设计\*\*: 完美适配移动端和桌面端

\- \*\*深色模式\*\*: 支持系统级深色主题

\- \*\*API 代理\*\*: 通过 PHP 代理安全调用火山引擎 API



\## 🚀 快速开始



\### 环境要求



\- PHP 7.4 或更高版本

\- Web 服务器（Apache/Nginx）

\- 火山引擎 API 密钥



\### 部署步骤



1\. \*\*克隆项目\*\*

&nbsp;  ```bash

&nbsp;  git clone https://github.com/yourusername/transly.git

&nbsp;  cd transly

&nbsp;  ```



2\. \*\*配置 API 密钥\*\*

&nbsp;  打开 `api\_proxy.php` 文件，将 `YOUR\_API\_KEY\_HERE` 替换为你的火山引擎 API 密钥：

&nbsp;  ```php

&nbsp;  $api\_key = 'your-actual-api-key-here';

&nbsp;  ```



3\. \*\*部署到服务器\*\*

&nbsp;  将项目文件上传到支持 PHP 的 Web 服务器目录。



4\. \*\*访问应用\*\*

&nbsp;  在浏览器中访问你的服务器地址，例如：`http://yourdomain.com/transly/`



\## ⚙️ 配置说明



\### API 密钥配置



推荐通过环境变量配置 API 密钥（更安全）：



```php

// 在 api\_proxy.php 中

$api\_key = getenv('VOLC\_API\_KEY') ?: 'your-fallback-key';

```



然后在服务器环境中设置 `VOLC\_API\_KEY` 环境变量。



\### 支持的模型



\- \*\*文本翻译\*\*: `deepseek-v3-250324`

\- \*\*图片 OCR 翻译\*\*: `doubao-seed-1-6-flash-250828`

\## 📱 使用说明



\### 文本翻译

1\. 在文本框中输入或粘贴要翻译的文本

2\. 选择源语言和目标语言

3\. 点击"翻译"按钮获取结果



\### 图片翻译

1\. 点击"导入图片"按钮或拖拽图片到页面

2\. 系统会自动识别图片中的文字并翻译

3\. 翻译结果会显示在输出区域



\### 功能按钮

\- \*\*⚡ 翻译\*\*: 执行翻译操作

\- \*\*📋 复制\*\*: 复制当前文本

\- \*\*✕ 清空\*\*: 清空输入/输出区域

\- \*\*⇄ 交换\*\*: 交换源语言和目标语言



\## 🔒 安全说明



\- API 密钥通过 PHP 代理保护，不会暴露在前端

\- 建议在生产环境中配置 CORS 白名单

\- 文件上传有大小限制（5MB）和类型验证

\- 所有 API 调用都经过验证和错误处理



\## 🐛 故障排除



\### 常见问题



1\. \*\*API 返回 401 错误\*\*

&nbsp;  - 检查 `api\_proxy.php` 中的 API 密钥配置

&nbsp;  - 确保火山引擎账户有足够的余额和权限



2\. \*\*图片上传失败\*\*

&nbsp;  - 检查图片大小是否超过 5MB

&nbsp;  - 确保图片格式为常见格式（JPG、PNG 等）

&nbsp;  - 检查 PHP 上传限制设置



3\. \*\*翻译结果不准确\*\*

&nbsp;  - 确认选择了正确的源语言

&nbsp;  - 对于长文本，建议分段翻译

&nbsp;  - 检查输入文本是否清晰可读



\### 错误日志

查看 Web 服务器的错误日志或浏览器控制台获取详细信息。
